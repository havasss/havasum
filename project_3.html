Branding / Responsive Website

Scandale

Scandale is a historic French lingerie brand that was founded in the 1930s. The brand was relaunched in 2021 with a new image that focuses on fashion and sustainability. As part of my job duties, I was responsible for rebranding, packaging design, and all online/offline graphic design. Additionally, I served as the UI/UX designer for their B2C responsive website, which was optimised for desktop, tablet, and mobile devices.
I conducted first-level research, developed user journeys, designed the web structure, created wireframes, conducted user testing, developed design systems, and created prototypes entirely on my own.
From Wireframe To Prototype
Once the clear website structure was established, the next step was to select a vendor to partner with for the programming job. During the programming period, it was important to prioritize quality control and time management to ensure timely delivery.
After the programming phase was complete, a small user test was conducted, and feedback from approximately 10 target audiences was gathered to improve the user experience.
